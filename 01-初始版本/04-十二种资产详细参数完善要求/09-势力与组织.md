# 【09-势力与组织】独身 .md 工业级参数设计规范

在长篇连载漫剧里，势力与组织是批量处理"群演（Extras）"、"背景杂兵"、"宗门守卫"以及"环境宏观美术"的核心。如果这一步空缺，大模型生图时，这一章的宗门弟子穿道袍，下一章的宗门弟子穿西装；或者是这一章的帮派杂兵拿着砍刀，下一章变成了激光剑。

成熟的漫剧管线会通过势力资产，强行规范"制服、组织图腾、大本营美术风格"，从而实现高效的群演和宏观美术控制。

当 Python 把势力名字（如"七玄门"、"野狼帮"）发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大电影级美术与声效参数模块**：

## 1. 基础组织架构 (Faction Baseline)

* **【势力/组织唯一名称】**：第一步盘点出的标准势力组织名。
* **【组织社会类别】**：明确归类。例如：隐世修仙宗门 / 江湖武林帮派 / 垄断巨头财阀 / 官方重案刑侦机构 / 废土拾荒联盟。
* **【核心代表/现任首领】**：前 10 章里出场的该组织核心人物（如：门主王绝楚、帮主贾天龙）。

## 2. 核心视觉图腾与徽章（生图的标志性 Prompt）

* **【图腾徽章图案描述】**：*（极其重要，用于生成旗帜、盾牌细节）* 组织的标志长什么样。例如：一团由三道卷云纹交织而成的烈火图案 / 泛着冷光的、由几何线条构成的双螺旋机械大厦剪影。
* **【徽章常态载体】**：这个标志通常印在哪里。例如：统一刺绣在制服的左胸口与大本营的战旗正中央 / 烫印在成员右侧肩膀的金属护甲片上。

## 3. 群体统一制式穿搭（控死群演/杂兵服装，防穿帮）

* **【核心高频色调】**：*（控制龙套群体的色彩）* 例如：主色调为深灰色，配以藏青色腰带 / 统一的黑色制式西装，内搭纯白衬衫。
* **【服装款式结构】**：例如：标准的宽袖右衽劲装长袍，束发，下身绑着黑色护腿布 / 贴身防弹战术背心，配黑色战术长裤与防暴头盔。
* **【武器/装备制式统一度】**：例如：普通弟子人人腰间悬挂一柄无名精钢长剑，剑鞘为黑色牛皮质地 / 守卫全员配备半透明的高能防暴光盾与短棍。

## 4. 大本营/据点建筑美术风格（宏观场景生图底座）

* **【整体建筑题材流派】**：例如：依山而建的古典中国宏伟宫殿群，白墙黑瓦 / 充满压迫感的哥特式重工业水泥巨构，外墙布满管道。
* **【环境氛围压迫感定调】**：例如：云雾缭绕，充满超凡脱俗的仙家出尘气象 / 昏暗阴森，空气中弥漫着廉价机油与血腥味的街头废土感。

## 5. 组织常态作风与队列气场（生视频的调度风格）

* **【群体高频姿态行为】**：例如：常态下每5人一组，手持长剑在山路巡逻，步伐整齐划一，神情警惕肃杀 / 散落聚集在酒馆角落，姿态松散，眼神凶狠，常有拍桌、拔刀的粗暴小动作。
* **【标志性组织口号/仪式】**：例如：见到高层时统一单膝跪地，双手抱拳横于胸前 / 冲锋时发出整齐的低吼。

## 6. 专属群体听觉重音（指导混音音效库调配）

* **【行进/出场群体音效】**：例如：数十人同时走动时，发出沉闷、极其整齐的布鞋踏地声"沙沙"声 / 沉重的皮靴重踏金属甲板声 / 越野摩托车疯狂轰鸣的排气声。
* **【群体背景环境白噪音】**：例如：练功场上百人整齐划一的"哈！喝！"练武喝喊声 / 机械流水线永不停歇的低频轰鸣与金属敲击声。

## 三、 极简拼装威力

到了第二阶段单章生图，如果小说正文写到："七玄门十几名巡逻弟子突兀地从山道浓雾中冲出，长剑瞬间出鞘"。

Python 后端直接提取 `七玄门.md` 里的：*"4-10名中国古代年轻武流武者，身穿交领紧袖深灰色劲装长袍，腰系黑色皮带，手持缠绕黑布条的精钢长剑，从怪石嶙峋的古典中式山道雾气中猛烈冲出，构图带有强烈的压迫感"*。

**这样出来的群演分镜，衣服的颜色、剑柄的线圈、山寨的建筑风格，直接100%卡死在原著的框架里。**
