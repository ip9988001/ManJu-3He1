# 【06-道具与线索】独身 .md 工业级参数设计规范

在漫剧制作中，道具（尤其是网文里的金手指、神兵利器、或者悬疑剧里的致命线索）往往会频繁出现**主观特写镜头**。如果一个法宝在远景里像个花瓶，到了特写镜头里变成了一个夜明珠，那就属于史诗级穿帮。

当 Python 把道具名字（如"神秘墨绿色小瓶"）发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大电影级视觉与声效参数模块**：

## 1. 基础资产标签 (Asset Baseline)

* **【道具唯一名称】**：第一步盘点出的标准道具名。
* **【资产核心类别】**：明确分类。例如：武器类（冷兵器/枪械） / 修仙法宝类 / 科技装备类 / 纯剧情线索物（如信件、带血的衣物）。
* **【当前归属/物理宿主】**：该道具目前常态下由哪位[角色]持有，或者存放在哪个[场景空间]。

## 2. 几何形态与体量感（生图 API 比例控制）

* **【绝对尺寸与参照物】**：*（严防生图模型把小瓶子画成水桶）* 必须提供具体的体量对比。例如：整体约巴掌大小，长约 12 厘米，刚好可以被一个 16 岁少年的右手完全握住。
* **【三维轮廓形状】**：例如：标准的扁平葫芦状 / 梯形长条状，边缘有不规则的锯齿。

## 3. 材质纹理与主色调（生图的质感控制）

* **【主色调与色彩渐变】**：例如：整体呈深邃的墨绿色，但在强光下瓶腹会透出淡淡的翠绿荧光。
* **【表面材质反光率】**：*（控制生图模型的材质高光）* 例如：介于玉石与青铜之间，表面毫无金属反光，呈现一种温润、内敛、带有岁月磨损的哑光质感。
* **【微观线索纹理/铭文】**：特写镜头必须看到的细节。例如：瓶身表面雕刻着极其细微的、类似树叶脉络的自然纹理，摸上去有凹凸感。

## 4. 超自然/科技特效表现（图生视频的动态核心）

* **【常态静止视觉】**：没有任何光效，看起来像个普通的凡物破瓶子。
* **【激活/高燃动态视觉（VFX）】**：*（留给视频 API 的动态大招提示词）* 例如：当置于月光下时，整个瓶身会开始半透明化，瓶体内部有绿色的流体像水波一样缓缓流动，并向外散发出半径 10 厘米的淡绿色微弱光晕。
* **【粒子与光束形态】**：例如：周围空气中的光点会凝聚成豆粒大小的绿色星芒，成螺旋状向瓶口汇聚。

## 5. 专属微观特写声效（指导配音音效库调配）

* **【物理碰撞拟音】**：例如：拔出瓶塞时发出清脆的"嘣"声 / 放在桌上时发出沉闷的玉石撞击声。
* **【超自然激活音效】**：例如：激活时伴随着极其轻微的、类似蜜蜂振翅般的"嗡嗡"低频共振声。

## 6. 战损痕迹与岁月感（精制作的细节抓手）

* **【表面瑕疵与损耗】**：例如：瓶底边缘有一处由于长年埋在泥土中产生的黑褐色泥垢，无法擦拭 / 崭新无瑕。

## 二、 落地：DeepSeek 最终吐出的 `06-道具与线索/神秘小绿瓶.md` 真实效果

按照你定死的纯文字暗号协议，生成的独立道具档案长这样：

```markdown
<道具参数开始>
道具唯一名称：神秘小绿瓶
资产核心类别：修仙法宝类
当前归属/物理宿主：由韩立随身贴胸口袋隐藏持有

绝对尺寸与参照物：整体约成人巴掌大小，高度约10厘米，瓶身最宽处约5厘米，刚好可以被一个少年的手掌完全包裹握住。
三维轮廓形状：标准的直颈、圆腹小药瓶形状，底部扁平可直立。

主色调与色彩渐变：主色调为极其深邃的墨绿色，但在月光或强光直射下，瓶腹中央会透出一抹如翡翠般的半透明翠绿。
表面材质反光率：非金非木，非石非玉，表面毫无金属刺眼反光，触手温润，呈现一种内敛、古朴的哑光质感。
微观线索纹理/铭文：瓶身表面雕刻着几道极其细微、繁复的叶片脉络纹理，纹路自然延伸至瓶颈，用指甲刮擦有明显的凹凸阻踩感。

常态静止视觉：在日光下毫无异状，表面暗淡无光，看起来与凡俗间普通的廉价泥瓶无异。
激活/高燃动态视觉（VFX）：当夜晚置于皎洁月光下时，整个瓶身会微微发热并开始半透明化，内部仿佛有碧绿色的液体在缓缓流动；瓶体周围会升腾起一层半径5厘米的淡绿色稀薄雾气。
粒子与光束形态：虚空中的月光光束会被瓶身强行吸附折射，无数豆粒大小的淡绿色荧光粒子会凭空出现在瓶子周围，呈螺旋状缓慢向瓶口汇聚。

物理碰撞拟音：拔出顶部的软木瓶塞时会发出清脆利落的"啵"一声；瓶身与坚硬地面碰撞时发出沉闷、清脆的玉石撞击声。
超自然激活音效：法宝凝聚月光粒子时，空气中会隐隐传出一种极其细微的、如水波荡漾般的低频"嘤嘤"共鸣声。

表面瑕疵与损耗：瓶底边缘的纹路缝隙里，粘连着一层长年埋在深山泥土中产生的、洗不掉的黑褐色陈年泥垢。
<道具参数结束>
```

## 三、 工业级道具卡逻辑

当第二阶段 Python 检索到这一章有"韩立端详小绿瓶"的镜头时，它把 `韩立.md` 的双手细节，配合 `神秘小绿瓶.md` 里的 *"高度约10厘米，墨绿色，哑光玉石质感，指尖触碰有叶片纹理"* 拼在一起发给 Flux 生图。

**这样出来的特写画面，瓶子的大小比例、质感、上面的泥垢和纹理，在大结局前就永远被死死锁定了。**
