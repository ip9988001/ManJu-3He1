# 【01-核心角色】独身 .md 细胞级参数设计规范

当 Python 把角色名字发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大解剖学与镜头对齐参数模块**，这是全自动管线中防换脸、防串音、控构图的灵魂底座：

## 1. 基础身份与生命周期坐标 (Identity & Phase Coordinates)

* **【角色唯一名称】**：第一步盘点出的标准名字（如：韩立）。
* **【当前生命周期阶段】**：*（网文升级流的命脉，控死当前10章的形态）* 例如：少年记名弟子期 / 青年筑基期 / 凡人童年期。
* **【生物性别属性】**：男 / 女 / 独立生物态。
* **【当前剧情阵营/社会身份】**：例如：七玄门神手谷记名弟子，地位卑微但隐蔽。

## 2. 电影级面部解剖学特征（生图 Flux / LoRA 控脸核心）

* **【皮表肤色与微观肌理】**：*（精确到光泽和质感）* 例如：常年暴晒导致的黝黑粗糙皮肤，无凡俗公子的白皙，额头有细微汗流痕迹，呈现健康的哑光质感。
* **【眼部解剖学与眼神光（Gaze）】**：*（直接决定角色黑化或正气）* 例如：标准的单眼皮、狭长眼型（单凤眼），瞳孔为深黑色，常态下眼神光极弱，透着一种远超同龄人的机警、戒备与极致的冰冷。
* **【发型结构与头饰追踪】**：例如：一头粗糙的黑色长发，没有任何发冠，仅用一根断裂的灰色草绳在脑后随意扎成马尾，额前留有几缕散乱的碎发。
* **【面部绝对一致性锚点（Face Lock）】**：*（给大模型下的死命令，防止多图换脸）* 例如：相貌极其平庸，毫无任何英俊或奇特标志，属于丢进流民堆里瞬间被蒸发的绝对路人脸。

## 3. 躯体形态与剪影轮廓（控制画面构图与比例）

* **【绝对身高与体型参照】**：例如：身高约 1.65 米，在同龄人中略显矮小。体型消瘦，但长期劈柴采药导致肩膀线条紧实，属于"穿衣显瘦、脱衣有肉"的柔韧身型。
* **【常态站姿与剪影轮廓（Silhouette）】**：例如：站立时身体习惯性微微前倾，重心下压，双臂自然下垂但双手随时处于可发力状态。

## 4. 动态神情与高频动作习惯（生视频微动/肢体控制）

* **【无意识微表情（Resting Face）】**：*（视频首帧动态）* 例如：习惯性嘴唇紧闭，嘴角微微向下抿，眉头轻微皱起，呈现出一种时刻在算计或防御的沉闷神态。
* **【高频肢体小动作】**：*（给 Luma/可灵的灵魂提示词）* 例如：当陷入思考或紧张时，右手大拇指会下意识地轻轻摩挲腰间的墨绿色小瓶 / 眼神会高频地向左下方斜视，快速打量周围环境。
* **【情绪爆发态肢体外显】**：例如：极度愤怒或动杀心时，面部肌肉反而完全僵硬死寂，眼神光瞬间消失，右侧太阳穴青筋微微跳动。

## 5. 专属声线解剖学与音轨参数（指导 TTS 声音克隆）

* **【声音生理年龄感】**：例如：16岁~18岁的变声期少年音，略带一丝沙哑。
* **【声带物理纹理（Texture）】**：例如：音色偏干、偏冷，毫无情绪波动，带有一种冷血、木讷、甚至有些机械的凡人木讷感。
* **【语速与长句停顿控制】**：例如：语速偏慢，每分钟约 180 字，说话时习惯在名词间进行短促停顿，字斟句酌，绝不吐露多余废话。

## 6. 局部特写抓手与镜头防错 (Close-up Anchors & Error Prevention)

* **【特写特异化镜头抓手】**：*（提示分镜台该拍他哪里）* 例如：一双因长期干粗活而布满老茧、指甲缝里带有草药黑汁的、粗糙的少年双手特写。
