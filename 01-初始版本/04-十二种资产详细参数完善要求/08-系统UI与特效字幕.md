# 【08-系统UI与特效字幕】独身 .md 工业级参数设计规范

在现代网文（尤其是全球爆火的系统流、数据流、第四天灾流、或者带有黑科技的科幻题材）中，"叮！系统面板开启"或者"功法熟练度+1"的视觉呈现，是刺激观众爽点的绝对核心。

如果这一步做塌了，大模型生图时会直接把这些数据大括号"焊死"在背景墙上，变成毫无质感的"背景小广告"。成熟的漫剧管线会将其定义为"独立图层挂件（Overlays）"，方便后续进行前后景分离或者动态特效渲染。

当 Python 把 UI 或字幕名字（如"长春功突破层数提示框"、"属性面板"）发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大电影级图层与动效参数模块**：

## 1. 基础资产标签 (Asset Baseline)

* **【UI/字幕唯一名称】**：第一步盘点出的标准 UI 或字幕名。
* **【界面呈现类别】**：明确归类。例如：全屏属性面板 / 局部弹窗提示框 / 虚空悬浮特效文字（无边框） / 视网膜 HUD 叠加。
* **【核心关联宿主】**：这个面板或文字主要是为了哪个[角色]、或者在哪种[技能/事件]触发时亮起的。

## 2. 美术风格与色彩滤镜（生图的 UI 美术定调）

* **【主画风题材】**：例如：半透明高科技全息荧光风格 / 古朴粗粝的暗金色隶书书法风格 / 羊皮纸卷轴魔幻风格。
* **【主色调与发光度（Glow）】**：*（控死生图色彩，严防大红大绿的土味特效）* 例如：主色调为幽蓝色（#00FFFF），带有微弱的霓虹发光边缘，整体透明度为 70%。
* **【边框与底衬纹理】**：例如：无任何实体边框，底衬为半透明黑色矩阵微粒 / 边框为布满青铜锈迹的古朴夔龙纹。

## 3. 核心文本内容与排版（图层叠加与文字生成依据）

* **【核心文本字样】**：大模型必须从原文中一字不差地抠出要显示的文字。例如："【系统提示：长春功突破至第四层！】"。
* **【文字排列与字体规范】**：例如：文字从上到下垂直排列，采用狂草书法字体 / 文字水平居中排列，采用发光像素电子屏字体。

## 4. 视网膜锚定与比例控制（画面构图占比）

* **【画幅空间锚定点】**：*（告诉渲染引擎这个面板挂在画面的哪个绝对位置）* 例如：居中悬浮于画面正中央 / 锚定在施法主角的头顶上方三分之一处 / 挂在画面四角（类似游戏机 HUD）。
* **【画幅视觉占比】**：例如：占据整个分镜画面的二分之一（强特写） / 仅占据画面顶部的十分之一。

## 5. 动态出场与消隐特效（图生视频的动态大招提示词）

* **【出场动效描述（VFX）】**：例如：面板伴随着一道极速划过的电子故障闪烁（Glitch Effect）凭空弹出 / 虚空文字如同墨汁在水里突然晕染开来一般显现。
* **【常态呼吸/粒子动态】**：例如：面板边缘有极其微弱的蓝色数据流从上往下像瀑布般滚动 / 文字周围有淡淡的金色灰烬粒子缓缓飘落。

## 6. 专属电声与拟音音效（指导音频库调配）

* **【弹出重音音效】**：例如：清脆的电子机械"叮"声 / 沉闷的远古洪荒丧钟鸣响声 / 尖锐的精神穿刺音。
* **【持续或消隐音效】**：例如：持续发出极其轻微的"滋滋"电流共振声 / 消失时伴随着气流散开的"呼"声。

## 二、 落地：DeepSeek 最终吐出的 `08-系统UI与特效字幕/长春功突破层数提示框.md` 真实效果

按照你定死的纯文字暗号协议，生成的独立 UI 档案长这样：

```markdown
<系统UI参数开始>
UI/字幕唯一名称：长春功突破层数提示框
界面呈现类别：局部弹窗提示框（悬浮于虚空）
核心关联宿主：触发于韩立闭关突破或功法更替之时

主画风题材：传统东方玄幻修仙风格，拒绝现代科技科技感。
主色调与发光度：主色调为深邃的古铜金与大理石白，文字自带一层微弱的淡金色佛光，毫无刺眼的霓虹杂色。
边框与底衬纹理：边框由细腻的、散发着玉石光泽的青竹纹理交织而成；底衬为一层半透明的墨色烟雾，隐约能透过底衬看到后面的背景。

核心文本字样：【系统提示：长春功成功突破至第四层！】
文字排列与字体规范：文字横向水平排列，字体为笔锋苍劲有力、带有大理石裂纹质感的金色隶书大字。

画幅空间锚定点：死死锚定在分镜画面的正中央，强制覆盖在男主角韩立的面部前方。
画幅视觉占比：横向拉满整个画面，纵向占据画面中央三分之一的黄金位置，形成强烈的视觉视觉冲击。

出场动效描述（VFX）：提示框并非生硬弹出，而在施法高潮瞬间，虚空中猛然炸开一圈淡金色的气流波纹，提示框由波纹中心瞬间放大并定格。
常态呼吸/粒子动态：金色隶书文字的笔画边缘，不断有极其细微的、由于功法突破产生出的淡绿色草木生机粒子（点点微光）向外逸散并凭空蒸发。

弹出重音音效：提示框震荡弹出的刹那，伴随着一声极为悠远、空灵、仿佛从远古寺庙中传来的重型铜钟敲击声"当——"。
持续或消隐音效：提示框存在期间，背景有极淡的、春风吹拂树叶的"沙沙"白噪音。
<系统UI参数结束>
```
