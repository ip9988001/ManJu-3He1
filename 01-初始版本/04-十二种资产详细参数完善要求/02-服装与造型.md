# 【02-服装与造型】独身 .md 细胞级参数设计规范

在长线漫剧管线中，衣服绝对不能只定一个"青衫"或"白袍"。大模型生图（Flux）在处理衣服时，最致命的缺陷就是**"款式结构大漂移"**（比如远景是右衽长袍，切到特写变成了对襟马褂）和**"材质塑料感"**。同时，视频模型（可灵/Luma）在让角色走动、打斗时，需要明确的**布料解算参数（物理垂坠感、摆动幅度）**，否则衣服就会像焊在皮肤上一样僵硬。

当 Python 把服装造型名词发给 DeepSeek 时，生成的独立 `.md` 文件必须锁死以下 **6 大材质与结构物理参数模块**：

## 1. 基础资产归属与时空绑定 (Asset Binding & Context)

* **【服装/造型唯一名称】**：第一步盘点出的标准服装名（如：恒岳派灰色记名弟子服）。
* **【专享角色/龙套群体】**：*（控死穿戴权限）* 明确本套衣服是特定[角色]的日常替换，还是某个[势力/组织]的统一制服。
* **【核心美术风格流派】**：例如：传统东方古典修仙禁欲系 / 江湖市井粗粝风 / 废土工业解构主义。

## 2. 面料微观学与光影解理（生图的 PBR 质感控制）

* **【主色调与色彩十六进制倾向】**：*（控死色差）* 精确描述色彩与渐变。例如：主色调为低饱和度的死灰色（类似#808080），在暗处偏向玄青，绝无任何亮眼反光。
* **【材质微观纹理与粗糙度】**：*（直接命令 Flux 消除塑料感）* 例如：由粗糙的麻砂布料纯手工编织，表面布满清晰的十字交叉经纬网格纹理，呈现完全吸光的粗粝哑光质感。

## 3. 三维裁剪与多图层防穿帮（控死款式结构拓扑）

* **【头部/束发造型对齐】**：*（头发和衣服是不可分割的造型）* 例如：无自带头部配件，但要求穿戴者的发型必须用一根同色系的灰色布带死死束成道髻。
* **【上身内外图层嵌套拓扑（Layering）】**：*（严防长袍变短衫的死命令）* 例如：采用交领右衽内衬（右侧衣襟掩向左腋下），外罩一件无袖的灰色交领长坎肩，前襟用两根布带系紧。
* **【下身裁剪与鞋履对接】**：例如：下身配同色系宽松束口长裤，裤脚死死塞进一双小腿高度的黑色粗布高帮靴内，靴底沾满干涸的黄色泥土。

## 4. 动态物理模拟反馈（图生视频的衣料解算灵魂）

* **【面料物理厚度与垂坠感】**：例如：面料厚重、生硬，常态下下摆自然垂直硬挺，抗风性极强，不易产生细碎褶皱。
* **【高频运动动态风阻反馈】**：*（给视频模型的大招）* 例如：当角色极速奔跑或施法时，衣袖与衣摆不会轻飘飘拂动，而是呈大块面地向后方猛烈鼓胀、猎猎作响，并在边缘拉出硬朗的折角褶皱。

## 5. 超自然/超科技微观特效表现（VFX 动态挂件图层）

* **【法阵/阵纹内敛视觉】**：针对仙家法衣或特殊防具。例如：常态下无任何特效；当注入灵力时，胸口处会隐隐浮现出一层半径10厘米的、散发着微弱白光的恒岳派太极防御阵纹。

## 6. 岁月磨损与历史痕迹（精制作的电影感细节）

* **【战损度与特定旧化痕迹（Weathering）】**：例如：双袖手肘处由于长期攀爬山岩有大面积的磨损发白起毛痕迹，左侧外摆下沿有一道约5厘米的长条形裂口，露出内衬的粗麻。

## 二、 落地：DeepSeek 最终吐出的 `02-服装与造型/恒岳派灰色记名弟子服.md` 真实效果

按照你定死的纯文字暗号协议，生成的细胞级服装档案长这样：

```markdown
<服装参数开始>
服装/造型唯一名称：恒岳派灰色记名弟子服
专享角色/龙套群体：归属于恒岳派所有底层记名弟子（如前期的王林、张虎）
核心美术风格流派：传统东方古典修仙底层粗粝风格

主色调与色彩十六进制倾向：整体由低饱和度的无光泽死灰色（#7F7F7F）统治，领口与袖口处拼接有更深一层的墨灰色麻布边框。
材质微观纹理与粗糙度：百分之百纯粗麻手工缝制，面料表面布满因工艺粗糙而产生的、肉眼可见的经纬编织十字颗粒纹理，对光线完全吸光，呈现极致的干枯、哑光粗粝质感。

头部/束发造型对齐：服装不含实体头饰，但强制穿戴该制服的角色必须将黑发用一根两指宽的灰色长布带在头顶正上方紧紧束成一个圆髻，发带两端垂在脑后。
上身内外图层嵌套拓扑：采用标准的古典中国交领右衽结构，双袖为窄袖（紧袖口设计，方便干活）。外侧套有一件无领、无袖的直筒灰色长坎肩，坎肩长度及膝，腰间系着一条手掌宽的藏青色粗麻腰带。
下身裁剪与鞋履对接：下身为同样死灰色的宽松灯笼长裤，裤脚在脚踝处被黑色的绑腿布死死缠绕六圈，并塞进一双千层底的黑色高帮粗布棉靴之中。

面料物理厚度与垂坠感：布料质地坚硬且厚重，常态站立时衣摆由于重力自然下垂，线条刚硬直折，不易产生现代丝绸类的轻盈软褶。
高频运动动态风阻反馈：在角色极速移动或施展引力术时，厚重的衣摆与长坎肩下摆会由于风阻呈大块面地向后方猛烈抛起翻滚，衣料拍击空气产生硬朗的动态剪影，彰显力量感。

法阵/阵纹内敛视觉：此为底层凡俗弟子服，体表不具备任何超自然灵力光效与流光特效。

战损度与特定旧化痕迹：整套衣服浆洗过度导致局部泛白。双袖手肘关节处由于长期劳作，布料严重磨损、起毛发干；腰带右侧由于长期悬挂砍柴刀，有一块被铁锈常年浸染而成的、洗不掉的暗褐色圆形污渍。
```

## 三、 工业级防变形逻辑

这一步把衣服的"嵌套拓扑（内长袍、外坎肩）"和"材质纹理（十字编织、完全吸光哑光）"卡死到这个深度，本地 Python 在第二阶段组合生成分镜时，发给 Flux 的词就拥有了绝对的控制力。

无论主角是正面、背面、还是拉到极远景，生图模型都必须乖乖按照 *"内穿灰色窄袖右衽长袍、外罩灰色长坎肩、脚穿高帮靴"* 的固定三维结构去渲染，**彻底解决了 AI 生成动漫长篇时"衣服每一帧款式都在变"的行业级灾难。**
