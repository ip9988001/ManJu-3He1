# 【10-非人生物与坐骑】独身 .md 工业级参数设计规范

在漫剧（特别是修仙、科幻、奇幻题材）中，妖兽、坐骑、机械生命或变异怪物的出场，往往是撑起视觉奇观和战斗张力的核心。如果这一步的数据定得含糊，生图模型（如 Flux）就会开始摆烂，直接给你生出一只普通的狼或普通的马，完全丧失了原著中那种"双头暴烈、浑身鳞片、散发煞气"的震撼感。同时，视频模型也需要精准的生物动态描述来控制它的肢体运动逻辑。

当 Python 把生物/坐骑名字（如"双头暴烈犬"、"铁奴/巨汉傀儡"）发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大电影级解剖与动态参数模块**：

## 1. 基础生物属性 (Biological Baseline)

* **【生物/坐骑唯一名称】**：第一步盘点出的标准生物名。
* **【生态/门类归属】**：明确分类。例如：修仙界一级低阶妖兽 / 废土辐射变异哺乳动物 / 赛博朋克重装战斗机械体 / 远古半透明幽冥灵体。
* **【常态物理宿主/持有者】**：该生物目前在原著里归谁控制，或者常态下栖息在哪个[场景空间]。

## 2. 解剖学形态与皮表细节（生图的核心控制 Prompt）

* **【皮表覆盖物材质与色调】**：*（控死皮肤质感，严防画成普通动物毛发）* 例如：全身无毛，覆盖着一层如黑铁般粗糙、坚硬的暗红色角质鳞片，边缘有骨质倒刺。
* **【头部五官与特异化器官】**：*（特写镜头的抓手）* 例如：长有两个完全相同的狰狞犬头；四只眼睛均呈现无瞳孔的猩红色，散发着嗜血的光芒；獠牙外露，嘴角不断滴落拉丝的、具有轻微腐蚀性的墨绿色涎水。
* **【四肢爪牙与攻击结构】**：例如：四足形似熊掌但大出三倍，前端生有长约20厘米的白骨弯钩利爪，深深扣入泥土中。

## 3. 绝对体量与比例控制（生图的构图比例）

* **【绝对尺寸与人机参照物】**：*（严防生图模型把巨兽画成哈巴狗）* 必须提供明确的参照对比。例如：整体体长约 3 米，肩高达 1.8 米，重逾千斤。当它四足站立时，其脊背高度刚好与一个成年人类的头部齐平。

## 4. 生态律动与行为特征（图生视频的天然动态 Prompt）

* **【常态静止/潜伏姿态】**：例如：前半身极度下压，后肢蓄力微屈，双头同时发出低沉的咆哮，尾部如同钢鞭般紧贴地面微微左右扫动。
* **【高频动态/奔袭姿态】**：*（视频 API 的灵魂动作词）* 例如：奔跑时四足交替贴地极速飞奔，身形在浓雾中拉出残影，双头交替向前做出疯狂撕咬的动作。

## 5. 超自然/科技常态气场（VFX 特效控制）

* **【常态体表能量辐射】**：例如：鳞片缝隙间常年往外溢散着一层极淡淡的、带有些许焦糊味的黑色煞气烟雾 / 机械关节处不断喷射出高压白雾粒子。

## 6. 专属音频咆哮纹理（指导配音音效库调配）

* **【发声频率与拟音质感】**：例如：发声时双头同时低吼，声音如同两块沉重的生铁在剧烈摩擦，沙哑、极其低沉，带有强烈的震动感。

## 二、 落地：DeepSeek 最终吐出的 `10-非人生物与坐骑/一级妖兽双头暴烈犬.md` 真实效果

按照你定死的纯文字暗号协议，生成的独立生物档案长这样：

```markdown
<非人生物参数开始>
生物/坐骑唯一名称：一级妖兽双头暴烈犬
生态/门类归属：修仙界一级低阶妖兽（犬类变异血统）
常态物理宿主/持有者：处于神手谷后山禁地密林中常态游荡

皮表覆盖物材质与色调：全身没有一根毛发，皮肤表面覆盖着一层如干涸血迹般的暗红色粗糙角质鳞片，鳞片在月光下闪烁着冰冷、粗粝的哑光。
头部五官与特异化器官：脖颈上方并排长有两个完全相同的狰狞犬头。两对犬眼均呈现出没有瞳孔的炽热猩红色；尖锐的獠牙交错外露，粗大的舌头耷拉在外，嘴角不断滴落带有恶臭的墨绿色涎水。
四肢爪牙与攻击结构：四肢粗壮如柱，脚掌异常宽大，前端生有五根长达15厘米、如同白骨打磨而成的弯钩巨爪，深深抓进地面的落叶堆里。

绝对尺寸与人机参照物：整体体长约3.2米（含尾部），背部肩高约1.7米。当它站在16岁少年韩立面前时，体型相当于三个韩立叠加大小，带有极强的泰山压顶般压迫感。

常态静止/潜伏姿态：前半身极度低垂贴地，龇牙咧嘴，两对猩红的眼睛死死盯着前方，粗壮的尾巴如同一根生锈的铁鞭，在身后的地面上焦躁地缓慢左右扫动。
高频动态/奔袭姿态：冲锋时四足猛烈蹬地，激起漫天泥土屑，庞大的身躯化为一道暗红色的闪电在密林中呈 Z 字形曲折极速飞奔，两个头颅交替向前做出极其凶狠的凌空撕咬动作。

常态体表能量辐射：暗红色鳞片的缝隙之间，常年若有若无地往外蒸腾着一缕缕极淡淡的黑色雾气（妖兽煞气），导致它周围的空气产生轻微的视觉模糊。

发声频率与拟音质感：两个头颅能同时发声，声音并非普通的狗叫，而是一种类似重型卡车发动机在低空轰鸣、又夹杂着钝器摩擦铁皮般的沉闷低频咆哮，能引发周围树叶轻微共振。
<非人生物参数结束>
```

## 三、 高精度调度逻辑

当第二阶段单章生产剧本，正文写到"林中突兀窜出一只一级妖兽双头暴烈犬，扑向韩立"时。

Python 后端直接把这个 `.md` 里的解剖学数据 *（长有两个狰狞犬头，猩红色无瞳孔眼睛，滴落墨绿色涎水，暗红色鳞片身躯，比人类大出三倍）* 提取出来，与场景 `密林.md`、人物 `韩立.md` 组合。

**生出来的画面，绝对是一个压迫感拉满的、原著里恐怖狰狞的变异妖兽，它的爪子、眼睛、体型比例在大结局前被死死锁住。**
