# 【11-世界设定与底座】独身 .md 工业级参数设计规范

在漫剧开发中，"世界设定（World Lore）"是最难具象化的一项。原著中往往充满大量抽象概念，比如"练气期、筑基期"、"灵气浓郁度"、"核心货币低阶灵石"。如果只让大模型生成一句话定义，到了生图和分镜阶段系统就会彻底抓瞎——**因为大模型根本不知道"筑基期修士"和"凡人"在画面上到底有什么视觉区别。**

当 Python 把世界设定名字（如"修炼境界-筑基期"、"核心货币-低阶灵石"）发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大电影级底层法则与视觉参数模块**：

## 1. 基础设定坐标轴 (Lore Baseline)

* **【设定/术语唯一名称】**：第一步盘点出的标准设定或术语名（如：修炼境界-筑基期）。
* **【世界观类别属性】**：明确分类。例如：修炼境界系统 / 基础世界法则（如天道誓言） / 宏观地理界域 / 核心能源与交易货币。
* **【核心规则底线（大白话定义）】**：*（强制灌输给大模型，死守战力底线，防止吃书）* 用一句话写明其核心逻辑（如：筑基期是修仙者的真正起点，可灵气液化，寿元两百载，战力对练气期形成绝对秒杀）。

## 2. 核心视觉显性化呈现（生图的核心控制 Prompt）

* **【常态外显视觉特征】**：*（极其重要，怎么向观众证明他是这个境界的强者）* 例如：筑基期修士周身常态环绕着一层肉眼可见的、极其稀薄的淡白色气旋护体；双脚可离地悬浮三尺，衣袍在无风状态下会自动向后微微猎猎飘动。
* **【全力爆发/越阶战斗视觉】**：例如：全力催动时，气旋瞬间实质化为一圈透明的圆形气浪向四周扩散，双眼瞳孔暴射出半寸长的实质性金色光芒。
* **【衰竭/破防状态视觉】**：例如：体表护体气旋瞬间如玻璃般碎裂瓦解，整个人由于重力狠狠砸回地面，头发散乱。

## 3. 能量流动与光影粒子（VFX 特效控制）

* **【能量主色调与光质】**：例如：主色调为纯净的大理石白，光质呈现半透明、带有雾状的流动感，拒绝刺眼的五彩杂光。
* **【粒子溢出与物理环流形态】**：例如：在施法高潮时，修士周围半径两米内，会有无数微小的、发光的晶莹水滴状粒子，违背重力缓缓向上空升腾。

## 4. 宏观环境与区域生态影响（场景/置景融合抓手）

* **【对周围死物的物理法则扭曲】**：例如：筑基期强者散发威压时，周围半径十米内的碎石屑、落叶会违背重力悬浮在半空中并剧烈颤抖；空气温度骤降，地面隐隐结出一层薄霜。

## 5. 结算流通过程规范（经济货币可视化特写）

* **【货币/能源物理长相】**：*（如果该术语是货币，控死特写长相）* 例如：低阶灵石为长约5厘米、规则的六角长条形晶体，整体呈半透明的淡蓝色，内部有雾状能量流转。
* **【交易结算动作定格】**：例如：交易时单手一挥，灵石在空中划过一道蓝色抛物线，稳稳落入对方掌心。

## 6. 专属听觉/规则共鸣调性（指导后期配音音效）

* **【规则激活/威压拟音描述】**：例如：境界威压释放瞬间，空气中伴随着一声极其沉闷的、类似重载卡车极速掠过时产生的"呜嗡"低频气流轰鸣声。

## 二、 落地：DeepSeek 最终吐出的 `11-世界设定与底座/修炼境界-筑基期.md` 真实效果

按照你定死的纯文字暗号协议，生成的独立设定档案长这样：

```markdown
<世界设定参数开始>
设定/术语唯一名称：修炼境界-筑基期
世界观类别属性：修炼境界系统
核心规则底线（大白话定义）：修仙者褪去凡胎、灵气液化的真正起点。寿元翻倍至两百载，其实力对体内的液态真元形成了对凡人或练气期修仙者的降维打击。

常态外显视觉特征：筑基期修士在静止时，身体周围会常态化环绕着一层由三道透明气流交织而成的淡白色气旋。角色身体可违背重力、毫无烟火气地离地悬浮约半米高，全身衣袍在无风的环境下依然会自发地向后轻轻摆动。
全力爆发/越阶战斗视觉：当修士全力催动真元时，体表的淡白色气旋会瞬间暴涨，向四周猛烈炸开一圈半径三米的圆形气浪；双眼的瞳孔深处会暴射出两道微小的、金色的实质化锋利光芒。
衰竭/破防状态视觉：当体内灵力枯竭或被击败时，周身的淡白色气旋会如实体玻璃般发出"啪"的清脆声响并碎裂瓦解，身体瞬间失去悬浮力，沉重地跌落回地面，沾染泥土。

能量主色调与光质：主色调为高纯度的乳白色与半透明水色，没有任何现代科技霓虹感，呈现出一种出尘的、古典仙家光质。
粒子溢出与物理环流形态：在高燃战斗或施法时，以修士为中心半径两米的虚空中，会凭空凝聚出无数豆粒大小、发光的晶莹液体微粒（液化灵气），这些微粒如同水滴逆流一般，缓缓自下而上向天空升腾飘散。

对周围死物的物理法则扭曲：筑基威压全开的瞬间，修士脚下半径五米内的枯叶、松软沙石会瞬间被无形重力强行震飞，清空出一片绝对干净的圆形地面；周围的空气由于真元剧烈震荡，在视觉上产生类似盛夏柏油路暴晒时的"热浪扭曲"折射。

货币/能源物理长相：非货币资产，本项无参数。
交易结算动作定格：非货币资产，本项无参数。

规则激活/威压拟音描述：威压或者护体气旋撑开的刹那，背景音里会突兀地切入一声极为沉闷、厚重、如同重型潜艇在深海中猛烈开机时发出的低频共振轰鸣声"轰嗡——"，用以反衬筑基期强者的绝对压迫感。
<世界设定参数结束>
```

## 三、 战力视觉代差逻辑

这套设计直接解决了大模型在画面创意上的"平庸"。

到了第二阶段单章生图，如果小说原文写到："韩立冷笑一声，筑基期的强横威压毫无保留地轰然散开！"

Python 后端直接把这个 `.md` 里的数据抠出来合并：*"16岁黝黑少年离地悬浮半米，衣袍无风自动，脚下半径五米枯叶被震飞，空气因热浪扭曲，周围有晶莹的水滴微粒违背重力向上空升腾"*。

**这样出来的战斗分镜，观众一眼就能看出什么是"筑基期神仙打架"，什么是"凡人互殴"，视觉代差和小说的爽点瞬间被狠狠拿捏。**
