# 【07-技能与特殊能力】独身 .md 工业级参数设计规范

在漫剧（尤其是修仙高燃、科幻异能流）中，技能释放是整部剧最吸睛、最容易拉高完播率的"大招高燃名场面"。如果技能参数写得笼统，大模型就会偷懒，千篇一律地给你画一些五颜六色的"魔法阵"或简单的"大火球"，这直接就把网文的精髓（如：斗气化马、踏雪无痕、水墨残影）给毁了。

当 Python 把技能名字（如"罗烟步"、"八极崩"）发给 DeepSeek 时，生成的独立 `.md` 文件必须死锁以下 **6 大电影级特效与剪辑参数模块**：

## 1. 基础能力坐标 (Ability Baseline)

* **【技能/能力唯一名称】**：第一步盘点出的标准技能名。
* **【超自然/技术类别】**：明确归类。例如：仙侠法术 / 武林外功功法 / 科技义体超频 / 精神异能。
* **【核心修习/施展者】**：该技能目前在前 10 章里主要由哪位[角色]施展。

## 2. 施展肢体与姿态控制（生图 API 动作控制）

* **【起手前摇肢体姿态】**：*（严防大模型把动作画呆滞）* 角色在释放瞬间的身体姿势。例如：沉肩坠肘，右拳向后极度蓄力拉满 / 双手在胸前疾速结出繁复手印 / 身体微微前倾，双腿重心压低。
* **【面部与神态突变】**：释放时的表情管理。例如：双眼由于能量充盈瞬间暴射出淡蓝色精光 / 咬紧牙关，额头青筋暴起 / 眼神冷漠深邃，毫无波动。

## 3. 核心能量视觉特效（生图的 VFX 核心 Prompt）

* **【能量主色调与光质】**：例如：主色调为如水墨般化开的死灰色，呈半透明、雾状，毫无刺眼强光。
* **【能量具象化形态】**：*（极其重要，控死大招长相）* 能量是以什么形状出现的。例如：缠绕在双腿周围的灰色稀薄烟雾 / 凝聚在拳头表面的实质化、带有丝丝电芒的深蓝色气旋。
* **【动态运行轨迹】**：例如：随着角色移动，灰色烟雾在空中留下一道长约两米的蛇形残影，随后迅速淡化散开。

## 4. 环境破坏与物理反馈（生视频与置景融合抓手）

* **【对周围死物的物理破坏】**：*（加深画面震撼感）* 例如：技能施展的瞬间，脚下的青石板因巨力寸寸开裂，碎石屑违背重力缓缓悬浮 / 踏雪无痕，地面不留任何脚印。
* **【对周围环境的空气/光线扰动】**：例如：由于极速移动，周围的草木被强风吹得疯狂向两边倒伏 / 拳头周围的空气由于高温产生剧烈的、类似柏油路暴晒的"热浪扭曲"。

## 5. 专属听觉重音与音效（指导音频库调配）

* **【启动与破空声效】**：例如：施展时伴随着极其沉闷的、类似闷雷在低空滚动的"轰隆"声 / 锐利刺耳的撕裂空气"嗤嗤"声。
* **【命中与炸裂声效】**：例如：击中物体时发出一声如金属撞击般清脆高亢的"铛"声 / 沉闷的肉体撞击声。

## 6. 能力等阶视觉代差（长线连载防吃书核心）

* **【当前等阶视觉限制】**：*（极其重要，主角刚学会和练到大满贯画面必须有代差）* 例如：当前为"初学入门"阶段，灰色烟雾仅能覆盖脚踝，且移动超过三步烟雾就会溃散 / "大圆满"时可身化浓烈黑烟。

## 二、 落地：DeepSeek 最终吐出的 `07-技能与特殊能力/罗烟步.md` 真实效果

按照你定死的纯文字暗号协议，生成的独立技能档案长这样：

```markdown
<技能参数开始>
技能/能力唯一名称：罗烟步
超自然/技术类别：武林轻功功法（带有一丝修仙界诡谲身法特质）
核心修习/施展者：韩立、厉飞雨

起手前摇肢体姿态：施展瞬间，角色身体突兀地向前极度倾斜，几乎与地面呈45度夹角，双腿在极短距离内高速交替，在后方拉出模糊的重影。
面部与神态突变：眼神极度冰冷专注，死死锁定移动方向，面部由于承受极速狂风吹拂导致黑发向后狂乱飞舞，嘴唇紧闭。

能量主色调与光质：主色调为毫无光泽的死灰色与半透明墨色，没有任何神圣的彩色光效，带有极强的阴冷、隐蔽感。
能量具象化形态：角色的双腿和脚踝周围，会凭空升腾起一缕缕如水墨画般在空气中晕染开来的灰色烟雾。
动态运行轨迹：随着角色身形忽左忽右地诡谲移动，那些灰色烟雾会在其移动轨迹上滞留约1秒钟，形成一条断断续续的、如长蛇般的灰色烟雾残影带。

对周围死物的物理破坏：踏雪无痕，草木不折，踩踏过的松软泥地或落叶堆上不会留下任何一丝脚印痕迹。
对周围环境的空气/光线扰动：虽然没有脚印，但由于身法极速移动，会带起一道肉眼可见的微弱透明气流风道，导致移动轨迹两侧的枯叶和杂草被狂风强行向两边吹开掀翻。

启动与破空声效：由于速度极快，身形启动的刹那，空气中会传出一声极其低沉、短促的、类似布匹被暴力撕裂的"嗤"的一声锐响。
命中与炸裂声效：属于移动身法，无直接命中声效。

当前等阶视觉限制：当前处于"初学第一层"阶段，灰色烟雾极为稀薄，且残影带长度无法超过两米，身形闪烁超过三次后，双腿肌肉会产生剧烈酸痛。
<技能参数结束>
```
