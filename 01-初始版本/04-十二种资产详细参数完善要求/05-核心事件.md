# 【05-核心事件】独身 .md 工业级参数设计规范

在漫剧最核心的就是**"画面张力"**。核心事件（Key Events）不是简单的剧情介绍，它在漫剧系统里其实是**"导演剪辑大戏的分镜大纲"**。

当 Python 把第一步盘点出的事件名字（如"意外捡到神秘绿瓶"、"厉飞雨悬崖对决"）发给 DeepSeek 时，DeepSeek 必须死磕原文，在这个事件的独立 `.md` 文件里吐出以下 **6 大硬核电影级参数模块**：

## 1. 基础事件坐标轴 (Event Baseline)

* **【事件唯一名称】**：第一步盘点出的标准事件名。
* **【发生绝对时空】**：*（强制关联已有资产，防止时空穿梭）* 必须指明是在哪个[场景空间]的哪个[具体时段]发生的。
* **【登场演员阵容】**：本场事件中在场、对峙的所有[角色名字]。

## 2. 电影级画面构图与镜头调度（生图 API 构图控制）

* **【远近景别与视角】**：明确该事件高潮镜头的景别。例如：特写（Close-up）/ 极远景（Extreme Wide Shot）/ 低角度仰拍（Low-angle Shot）/ 角色第一人称主观视角（POV）。
* **【演员站位与空间对峙（Blocking）】**：*（极其重要，生图模型最怕分不清左右和距离）* 必须明确角色之间的物理空间关系。例如：A 站在画面左侧三分之一处，身体前倾；B 站在右侧背景的暴雨中，两人相距五米，形成强烈的对角线构图。
* **【背景环境物象突变】**：事件爆发时，周围环境发生了什么视觉变化。例如：原本平静的雾气因掌风大开突然向四周疯狂散开 / 窗外划过一道刺眼的闪电，照亮暴雨中的竹林。

## 3. 演员微表情与肢体定格（生视频首帧控制）

* **【核心角色 A 的微表情】**：瞳孔地震、咬牙切齿、面部肌肉抽搐、眼神闪过一丝极度惊恐。
* **【核心角色 A 的肢体定格】**：右手死死捂住流血的腹部，左手单指指向前方，双腿因脱力微微屈膝。
* **【核心角色 B 的神态肢体】**：居高临下地俯视，嘴角挂着一抹残忍的冷笑，右手倒提着一把正往下滴血的黑铁长刀。

## 4. 光影氛围与粒子特效突变（VFX 视觉抓手）

* **【核心光源异变】**：事件进入高潮时，光线发生的戏剧化反差。例如：原本昏暗的房间被法宝激发的强光瞬间吞噬 / 月光突然被乌云遮挡，画面整体陷入极具压迫感的暗调。
* **【超自然/科技粒子动态】**：例如：空气中凭空爆发出无数密密麻麻的淡蓝色闪电火花 / 四周的时间仿佛静止，无数泥土碎屑悬浮在半空中。

## 5. 视听重音与快剪卡点（指导音频库与 FFmpeg 剪辑）

* **【瞬间视听重音（Audio Hit）】**：原文中暗示的、最大的一声巨响。例如：刀剑猛烈撞击的刺耳金属摩擦声 / 身体沉重倒地的沉闷撞击声 / 闷雷在头顶炸裂声。
* **【视觉剪辑卡点建议】**：提示剪辑台在这里该怎么切镜头。例如：建议在此处进行 0.5 秒的画面剧烈颤动（模拟冲击波）/ 建议画面瞬间切为纯黑 0.1 秒再亮起（模拟被击晕）。

## 6. 剧情熔断与悬念留钩（短剧连续剧的命脉）

* **【单集熔断定格画面】**：*（如果这一章恰好在这里结束，画面该怎么定格）* 例如：定格在黑铁长刀距离男主喉咙仅剩一公分的极度悬疑特写画面。

## 二、 落地：DeepSeek 最终吐出的 `05-核心事件/意外捡到神秘绿瓶.md` 真实效果

按照你定死的纯文字暗号协议，生成的独立事件详尽卡片长这样：

```markdown
<核心事件参数开始>
事件唯一名称：意外捡到神秘绿瓶
发生绝对时空：场景空间（神手谷药田草丛） | 昼夜时段（时段-深夜）
登场演员阵容：韩立（少年期）

远近景别与视角：特写（Close-up）配合低角度仰拍，视线聚焦在主角的双手和地面。
演员站位与空间对峙：韩立整个人半蹲在画面中央，身体极度前倾，双手呈拨开杂草的姿态。画面下四分之一处为泥土与草根。
背景环境物象突变：四周原本静止的浓密白雾，随着主角拨开草丛的动作，在画面边缘产生轻微的涡流翻滚。

核心角色 A 的微表情：双眼瞬间睁大，瞳孔剧烈收缩（瞳孔地震），嘴唇微张，眼神中交织着不可思议与极度的好奇。
核心角色 A 的肢体定格：右手的食指与中指指尖已经触碰到了泥土中埋藏的异物，左手死死按住旁边的泥地，浑身肌肉由于紧张而紧绷。
核心角色 B 的神态肢体：本场无其他角色。

核心光源异变：皎洁的银白色月光原本均匀洒在山谷中，在主角指尖触碰瓶身的瞬间，月光似乎在瓶身表面产生了奇妙的折射聚焦，形成一个极小的、高亮的银白光斑。
超自然/科技粒子动态：瓶身周围的草叶上，开始凭空升腾起几颗微小的、散发着幽绿荧光的半透明粒子，悬浮在空气中。

瞬间视听重音：主角急促、沉重的呼吸声，以及指甲触碰到非金非玉硬物时，发出的一声极其清脆的"嗒"的异响。
视觉剪辑卡点建议：在指尖触碰瓶子的瞬间，建议全屏进行一帧（0.03秒）的"闪白"效果，随后画面剧烈颤动，模拟主角内心受到的震撼。

单集熔断定格画面：画面死死定格在黝黑少年的指尖刚刚触碰到墨绿色瓶身、几颗绿色微粒悬浮在半空中的定格特写，剧情戛然而止。
<核心事件参数结束>
```

## 三、 工业级分镜拼装逻辑

这样提炼出来的 `.md` 文件，参数不仅详细到了牙齿，而且全部是**直接面向画面、面向声音、面向镜头语言的工业参数**。

后面第二阶段 Python 带着单章 TXT 正文和这个 `.md` 文件去调 API 时，大模型闭着眼睛都能把 *"Low-angle close-up, a 16-year-old tan-skinned Chinese boy crouching in misty field at night, sharp moonlight focusing on his fingers touching a mysterious green bottle, cliffhanger tension..."* 这种神级英文提示词给直接拼出来。
