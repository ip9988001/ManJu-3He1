## 二、素材下载与缓存

### 2.1 功能

把云端生成的图片、视频、音频下载到本地缓存目录。

### 2.2 实现

```python
class AssetDownloader:
    def __init__(self, cache_dir="cache"):
        self.cache_dir = cache_dir
        os.makedirs(cache_dir, exist_ok=True)

    async def download(self, url: str, file_type: str):
        filename = url.split("/")[-1]
        path = os.path.join(self.cache_dir, filename)
        async with aiohttp.ClientSession() as session:
            async with session.get(url) as resp:
                with open(path, "wb") as f:
                    f.write(await resp.read())
        return path
```

## 三、静态图 → 动态视频（Ken Burns 效果）

### 3.1 功能

让静态图片"动起来"。解决"AI 图片不会动"的问题。

### 3.2 效果类型

| 画幅 | 默认运动方向 | 说明 |
|------|-------------|------|
| 9:16 竖屏 | 纵向平移 | 以纵向缓慢平移为主 |
| 16:9 横屏 | 横向平移 | 以横向缓慢平移为主 |
| 1:1 方形 | 缓慢缩放 | 以居中构图的缓慢缩放为主 |

### 3.3 实现

```python
class ImageAnimator:
    def ken_burns(self, image_path, duration, output, axis="vertical"):
        if axis == "vertical":
            vf = "zoompan=z='min(zoom+0.0015,1.5)':d=1:x='iw/2-(iw/zoom/2)':y='ih/2-(ih/zoom/2)'"
        elif axis == "horizontal":
            vf = "zoompan=z='min(zoom+0.001,1.3)':d=1:x='if(eq(on,1),iw/2,iw/2-(iw/zoom/2)+on*0.5)':y='ih/2-(ih/zoom/2)'"
        else:  # zoom
            vf = "zoompan=z='min(zoom+0.002,1.5)':d=1:x='iw/2-(iw/zoom/2)':y='ih/2-(ih/zoom/2)'"
        
        cmd = ["ffmpeg", "-loop", "1", "-i", image_path, "-t", str(duration),
               "-vf", vf, "-c:v", "libx264", "-pix_fmt", "yuv420p", output]
        subprocess.run(cmd, check=True)
        return output
```

## 六、ASS 字幕生成

### 6.1 功能

根据台词生成 ASS 格式工业级字幕。字幕位置根据画幅自动调整。

### 6.2 画幅-字幕位置映射

| 画幅 | 字幕位置（画面高度百分比） |
|------|------------------------|
| 9:16 | 80% |
| 16:9 | 88% |
| 1:1 | 82% |

### 6.3 实现

```python
class SubtitleGenerator:
    def create_ass(self, dialogue, output, subtitle_position=0.80):
        ass_header = """[Script Info]
Title: AI Drama
ScriptType: v4.00+

[V4+ Styles]
Style: Default,Arial,40,&H00FFFFFF,&H00000000,&H00000000,&H66000000

[Events]
"""
        events = ""
        for i, line in enumerate(dialogue.split("\n")):
            events += f"Dialogue: 0,0:00:{i:02d}.00,0:00:{i+2:02d}.00,Default,,0,0,0,,{line}\n"
        with open(output, "w", encoding="utf-8") as f:
            f.write(ass_header + events)
        return output
```

### 7.2 流程

2. 图片动画化（Ken Burns）
3. 下载配音
4. 音画合成
5. 叠加字幕

```python
class VideoSynthesizer:
    async def synthesize_scene(self, scene):

        video = self.animator.ken_burns(image, scene["duration"], "temp_scene.mp4")
        audio = await self.downloader.download_audio(scene["audio_url"])
```

## 八、整集拼接

### 8.1 功能

一集所有分镜片段按顺序拼接，处理转场效果（淡入淡出、切黑等）。

### 8.2 实现

```python
class EpisodeAssembler:
    def assemble(self, scene_files, output):
        with open("concat.txt", "w") as f:
            for s in scene_files:
                f.write(f"file '{s}'\n")
        cmd = ["ffmpeg", "-f", "concat", "-safe", "0", "-i", "concat.txt", "-c", "copy", output]
        subprocess.run(cmd, check=True)
        return output
```

## 十、WebUI 渲染状态联动

### 10.1 功能

【取自系统2】WebUI 根据料单中的状态字段决定显示内容。

| 状态 | WebUI 显示 | 用户可操作 |
|------|-----------|-----------|
| 未渲染 | 灰色虚线框占位符 | 可点击"一键生成" |
| 渲染中 | 菊花转圈图 | 等待 |
| 已封顶 | 精美成品图 | 可点击"Re-roll"重跑 |

### 10.2 状态驱动流程

1. 用户打开 WebUI → 前端读取 `.md` 文件
2. 看到节点 01 是"已封顶" → 加载 `image/frame_01.png` 展示
3. 看到节点 02 是"渲染中" → 显示转圈
4. 看到节点 03 是"未渲染" → 显示灰色占位符

### 10.3 单镜 Re-roll

1. 用户点击镜头 02 的"Re-roll"
2. Python 定位到 `### [生图控制节点_02]`
3. 改状态为"渲染中"
4. 重新打包发送请求
5. 新图回来覆盖 `frame_02.png`
6. 状态改回"已封顶"
7. 网页卡片一闪，新面孔换上去，镜头 01 和 03 不受影响

## 十一、Batch 批量生产

### 11.1 功能

批量处理多个分镜，自动合成整集。

```python
class BatchProducer:
    async def produce_episode(self, scenes):
        scene_outputs = []
        for scene in scenes:
            video = await self.synthesizer.synthesize_scene(scene)
            scene_outputs.append(video)
        return self.assembler.assemble(scene_outputs, "episode.mp4")
```
