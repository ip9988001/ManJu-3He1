## 一、十章视听控制总纲

### 1.1 功能说明

【取自系统2】每 10 章为一轮，生成当前段戏的"分场导演手册"。它锁定了这 10 章内的剧情里程碑、战力流转、情绪曲线和最后一幕定格画面，是微观单章生产的直接指挥文件。

### 1.2 输入

| 输入 | 来源 | 作用 |
|------|------|------|
| 宏观灯塔 | 当前百章航海图 | 提供长线因果、伏笔暗线约束 |
| 上一期总纲 | 上一轮十章总纲 | 提供情绪、断点、状态的代际传递 |
| 原始生肉 | 第N至N+9章原文 | 提供基础故事文本 |

### 1.3 字数流控

$$W_{in} \times 15\% \le W_{out} \le W_{in} \times 20\%$$

- 4 万字输入 → 输出 6,000-8,000 字
- 低于 15% 熔断重跑，高于 20% 强制截断

### 1.4 四大维度详解

#### 维度一：分章高密度剧情里程碑（占 40%）

| 字段 | 说明 |
|------|------|
| `[Chapter_X_Milestone]` | 逐章核心推进任务，只准输出物理行为，拒绝文学评论 |
| `[Episodic_Cut_&_Hooks]` | 将该章切分为若干漫剧单集（每集 60-80 秒），每集包含：|
| `[Text_Hook]` | 文本留钩文案（主角在第几句台词时突然中断留下悬念） |
| `[Visual_Hook]` | 画面定格钩子（特写，如一张沾满血迹的传音符在虚空中自燃） |

#### 维度二：核心线索与战力流转图（占 20%）

| 字段 | 说明 |
|------|------|
| `[Chapter_Power_Monitor]` | 逐章监控并更新全场角色的真实战力与状态改变 |
| `[Prop_Physical_Host_Tracking]` | 卡死道具的物理宿主和流转轨迹 |

**道具追踪标准格式：**
```
章节号 | 道具名 | 当前物理持有者 | 空间存放位置 | 表面物理形态与战损特效
第15章 | 天逆珠 | 王林 | 右侧裤腿内夹层缝合处 | 珠体表面产生细微皲裂，吸收水汽时散发淡淡白雾
```

#### 维度三：全局情绪曲线与视听基调（占 20%）

| 字段 | 说明 |
|------|------|
| `[Emotional_Vibration_Curve]` | 以单章为节点，情绪震荡数值 -5（极度屈辱绝望）到 +5（极度高燃反杀） |
| `[Cinematic_Color_Grading_Commands]` | 电影级调色指令，必须精确指定 Hex 色域与光影参数 |
| `[BGM_Arrangement_Shifts]` | 配乐变阵指令，定死音轨主导乐器与频响质感 |

**调色指令示例：**
```
[主调色盘: #1C2321 至 #3E504B（阴冷水墨绿）
| 光影刚性: 高反差、强侧逆光、人物边缘留出亮线白边
| 环境粒子: 细密暗流雾气，空气动力学流速 0.2m/s]
```

**BGM 变阵示例：**
```
移除高音竹笛，转由低音大提琴拉奏长单音，伴随不规律的高频金属摩擦声，烘托危机感
```

#### 维度四：本期最后一幕视听定格（占 20%）

| 字段 | 说明 |
|------|------|
| `[Zero_Loss_Welding_Point]` | 用纯物理、客观的电影镜头语言死死锁定制片最后一镜 |

**必须包含：**
- 构图站位：`王林位于画面左侧1/3处，身体前倾30度，右手扣住储物袋。`
- 核心光源：`唯一主光源为头顶上方45度落下的冰冷月光，面部留出大面积阴影。`
- 物理粒子：`四周有散落的碎石处于慢动作悬浮状态。`
- 环境白噪音：`低频风声（120Hz）并夹杂衣袍剧烈猎猎作响的音频。`

### 1.5 红线约束

| 红线 | 说明 |
|------|------|
| 字数流控 | 低于 15% 熔断，高于 20% 截断 |
| 调色必须精确 | 必须给出 Hex 色值 |
| 最后一幕必须物理描述 | 禁止主观描述，必须是可执行的镜头语言 |
| 道具追踪必须精确 | 必须写明物理持有者和存放位置 |

### 1.6 人工审核点

✅ 必须审核——确认剧情里程碑、情绪曲线、调色方案、最后一幕定格。

## 二、十二大资产细胞级参数（前四类）

### 2.0 通用设计规范

**【取自系统2】** 每类资产都有 6 大标准参数模块，面向 AI 生图/视频接口，用纯中文尖括号标签输出。

**6 大通用模块框架：**

| 模块编号 | 模块名称 | 通用功能 |
|----------|----------|----------|
| MODULE_01 | 基础识别与坐标 | 名称、类别、归属、生命周期阶段 |
| MODULE_02 | 物理三维解构 | 面部解剖/材质纹理/空间布局（生图核心） |
| MODULE_03 | 动态环境反馈 | 风阻运动、光质折射、高速形变（视频核心） |
| MODULE_04 | 剧情摩擦与战损 | 版本号、破损痕迹、消耗暗淡度 |
| MODULE_05 | 音频克隆与声效 | 声带物理材质、爆破分贝、金属频宽 |
| MODULE_06 | 后期卡点参数 | FFmpeg 闪白帧数、震屏抖动率、转场指令 |
